# pyAmpli dependencies

## Python
Normally these dependencies are installed automatically when running setup.py
#### Python 2.7 or later
#### pysam (v0.8.4) (http://pysam.readthedocs.io/)
     pip install pysam
      
#### pyvcf (v0.6.8) (http://pyvcf.readthedocs.io/)
     pip install pyvcf
     
## System
#### samtools v1 or later
     apt-get samtools
     
#### pigz
     apt-get pigz
